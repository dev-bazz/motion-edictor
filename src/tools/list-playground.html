technique guide
📌::marker — fast and clean. Supports only color, font-size, font-weight, content. No bg/padding/radius.
🔧::before — full box model. Set list-style:none, use display:flex on li, style the pseudo-element freely.
🖼️img icon — three sub-approaches: list-style-image (no size ctrl), ::before url() (sized), or inline <img> (max control).
